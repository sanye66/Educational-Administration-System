# 接口调用规范

本文档定义了教学管理系统中内部服务调用与外部 API 调用的规范。

---

## 1. Feign 客户端规范

### 1.1 定义接口

```java
@FeignClient(name = "edu-user", path = "/api/user")
public interface UserFeignClient {

    /**
     * 根据ID获取用户
     */
    @GetMapping("/{id}")
    R<User> getUserById(@PathVariable("id") Long id);

    /**
     * 根据用户名获取用户
     */
    @GetMapping("/username/{username}")
    R<User> getUserByUsername(@PathVariable("username") String username);

    /**
     * 新增用户
     */
    @PostMapping
    R<Long> saveUser(@RequestBody User user);
}
```

### 1.2 统一响应格式

所有接口统一使用 `R<T>` 包装响应：

```java
@Data
public class R<T> implements Serializable {
    private int code;
    private String message;
    private T data;

    public static <T> R<T> ok() {
        return restResult(null, SuccessCode.SUCCESS);
    }

    public static <T> R<T> ok(T data) {
        return restResult(data, SuccessCode.SUCCESS);
    }

    public static <T> R<T> failed(ServiceException exception) {
        return restResult(null, exception.getErrorCode());
    }

    private static <T> R<T> restResult(T data, ErrorCode errorCode) {
        R<T> apiResult = new R<>();
        apiResult.setCode(errorCode.getCode());
        apiResult.setMessage(errorCode.getMessage());
        apiResult.setData(data);
        return apiResult;
    }
}
```

### 1.3 超时配置

```yaml
feign:
  client:
    config:
      default:
        connectTimeout: 5000
        readTimeout: 10000
      edu-user:
        connectTimeout: 3000
        readTimeout: 5000
```

## 2. 服务间调用规范

### 2.1 调用流程

```java
@Service
public class CourseServiceImpl implements ICourseService {

    private final UserFeignClient userFeignClient;

    public CourseServiceImpl(UserFeignClient userFeignClient) {
        this.userFeignClient = userFeignClient;
    }

    @Override
    public CourseDetailVO getCourseDetail(Long courseId) {
        Course course = getById(courseId);
        
        // 调用用户服务获取教师信息
        R<User> teacherResult = userFeignClient.getUserById(course.getTeacherId());
        if (teacherResult.getCode() != 0) {
            throw new ServiceException("获取教师信息失败");
        }
        
        // 构建返回对象
        return buildCourseDetail(course, teacherResult.getData());
    }
}
```

### 2.2 降级处理

使用 Hystrix 或 Sentinel 实现服务降级：

```java
@FeignClient(name = "edu-user", fallback = UserFeignClientFallback.class)
public interface UserFeignClient {
    @GetMapping("/{id}")
    R<User> getUserById(@PathVariable("id") Long id);
}

@Component
public class UserFeignClientFallback implements UserFeignClient {
    private static final Logger LOGGER = LoggerFactory.getLogger(UserFeignClientFallback.class);

    @Override
    public R<User> getUserById(Long id) {
        LOGGER.warn("UserFeignClient getUserById 降级处理, id: {}", id);
        return R.failed(new ServiceException("服务暂时不可用"));
    }
}
```

## 3. 内部服务调用

### 3.1 直接注入 Service

同一服务内直接注入调用：

```java
@Service
public class StudentServiceImpl implements IStudentService {

    private final IGradeService gradeService;

    public StudentServiceImpl(IGradeService gradeService) {
        this.gradeService = gradeService;
    }

    @Override
    public StudentDetailVO getStudentDetail(Long studentId) {
        Student student = getById(studentId);
        List<Grade> grades = gradeService.getGradesByStudentId(studentId);
        
        return buildStudentDetail(student, grades);
    }
}
```

### 3.2 跨服务调用

使用 Feign Client 进行跨服务调用：

```java
@Service
public class ExamServiceImpl implements IExamService {

    private final CourseFeignClient courseFeignClient;
    private final ClassroomFeignClient classroomFeignClient;

    public ExamServiceImpl(CourseFeignClient courseFeignClient,
                          ClassroomFeignClient classroomFeignClient) {
        this.courseFeignClient = courseFeignClient;
        this.classroomFeignClient = classroomFeignClient;
    }
}
```

## 4. 外部 API 调用

### 4.1 RestTemplate 使用

```java
@Service
public class ExternalApiService {

    private final RestTemplate restTemplate;

    public ExternalApiService(RestTemplate restTemplate) {
        this.restTemplate = restTemplate;
    }

    public ExternalData getExternalData(String param) {
        String url = "https://api.example.com/data?param={param}";
        return restTemplate.getForObject(url, ExternalData.class, param);
    }
}
```

### 4.2 WebClient 使用（推荐）

```java
@Configuration
public class WebClientConfig {

    @Bean
    public WebClient webClient() {
        return WebClient.builder()
                .baseUrl("https://api.example.com")
                .defaultHeader(HttpHeaders.CONTENT_TYPE, MediaType.APPLICATION_JSON_VALUE)
                .build();
    }
}

@Service
public class ExternalApiService {

    private final WebClient webClient;

    public ExternalApiService(WebClient webClient) {
        this.webClient = webClient;
    }

    public Mono<ExternalData> getExternalDataAsync(String param) {
        return webClient.get()
                .uri("/data?param={param}", param)
                .retrieve()
                .bodyToMono(ExternalData.class);
    }
}
```

## 5. 调用链规范

### 5.1 链路追踪

所有服务间调用必须传递链路追踪信息：

```java
@Component
public class FeignRequestInterceptor implements RequestInterceptor {

    @Override
    public void apply(RequestTemplate template) {
        // 传递 TraceId
        String traceId = TraceContext.getTraceId();
        if (StringUtils.isNotBlank(traceId)) {
            template.header("X-Trace-Id", traceId);
        }
        
        // 传递用户信息
        String userId = SecurityUtils.getUserId();
        if (StringUtils.isNotBlank(userId)) {
            template.header("X-User-Id", userId);
        }
    }
}
```

### 5.2 日志规范

```java
@Service
public class UserServiceImpl implements IUserService {

    private static final Logger LOGGER = LoggerFactory.getLogger(UserServiceImpl.class);

    @Override
    public User getUserById(Long id) {
        LOGGER.info("开始获取用户信息, userId: {}", id);
        long startTime = System.currentTimeMillis();
        
        try {
            User user = userMapper.selectById(id);
            LOGGER.info("获取用户信息成功, userId: {}, 耗时: {}ms", id, 
                       System.currentTimeMillis() - startTime);
            return user;
        } catch (Exception e) {
            LOGGER.error("获取用户信息失败, userId: {}", id, e);
            throw new ServiceException("获取用户信息失败");
        }
    }
}
```

## 6. 错误处理

### 6.1 统一错误码

| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 1000-1999 | 系统错误 |
| 2000-2999 | 认证错误 |
| 3000-3999 | 参数错误 |
| 4000-4999 | 业务错误 |
| 5000-5999 | 第三方服务错误 |

### 6.2 错误处理示例

```java
@FeignClient(name = "edu-user", fallbackFactory = UserFeignClientFallback.class)
public interface UserFeignClient {
    @GetMapping("/{id}")
    R<User> getUserById(@PathVariable("id") Long id);
}

@Component
public class UserFeignClientFallback implements FallbackFactory<UserFeignClient> {
    @Override
    public UserFeignClient create(Throwable cause) {
        return id -> {
            if (cause instanceof FeignException.NotFound) {
                return R.failed(new ServiceException(4001, "用户不存在"));
            }
            if (cause instanceof FeignException.ServiceUnavailable) {
                return R.failed(new ServiceException(5001, "服务暂时不可用"));
            }
            return R.failed(new ServiceException(5000, "调用用户服务失败"));
        };
    }
}
```
