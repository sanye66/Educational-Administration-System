# Harness Changes

This directory tracks change sources and event configurations for the Educational Administration System.

## Overview

Changes define how Harness monitors and responds to code changes, deployments, and infrastructure modifications.

## Structure

```
changes/
├── sources/            # Change source configurations
├── triggers/           # Pipeline trigger definitions
└── events/             # Event handling configurations
```

## Usage

Configure change sources to automatically trigger pipelines on code commits, image updates, or infrastructure changes.
