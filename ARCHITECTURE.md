### 服务间通信

1. **同步调用**: OpenFeign
2. **异步消息**: RocketMQ
3. **服务发现**: Nacos
4. **负载均衡**: Spring Cloud LoadBalancer

## 安全架构

### 认证流程

```
┌──────────┐         ┌──────────┐         ┌──────────┐
│  Client  │         │ Gateway  │         │  Auth    │
└────┬─────┘         └────┬─────┘         └────┬─────┘
     │                    │                    │
     │  1. Login Request  │                    │
     │───────────────────>│                    │
     │                    │  2. Forward        │
     │                    │───────────────────>│
     │                    │                    │ 3. Verify Credentials
     │                    │                    │ 4. Generate JWT
     │                    │                    │
     │                    │  5. Return Token   │
     │                    │<───────────────────│
     │  6. Return Token   │                    │
     │<───────────────────│                    │
     │                    │                    │
     │  7. Request with JWT                     │
     │───────────────────>│                    │
     │                    │  8. Validate JWT   │
     │                    │───────────────────>│
     │                    │                    │
     │                    │  9. Valid          │
     │                    │<───────────────────│
     │                    │                    │
     │                    │  10. Route to Service
     │                    │──────────────────────────> Service
```
